React Native / Expo SDK for the Wootangular Hive.

**For Lilian. For Lily.**
Named for the flower that grows in the swamp.

## Install
```
npm install lilypod-rn
# or
yarn add lilypod-rn
```

## Usage
```javascript
import { LilypodClient, useFuse, useHiveState, HiveStatus, BOOL_NULL } from 'lilypod-rn';

const client = new LilypodClient('https://wootangular369.up.railway.app');

// Hook
const { fuse, result, loading } = useFuse(client);

// Component
<HiveStatus hiveState={BOOL_NULL} totalHeat={74.0} totalEntropy={12} />
```
